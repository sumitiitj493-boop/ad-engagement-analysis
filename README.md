# Instagram User Behavior Analysis

## Overview
An end-to-end data analysis project exploring commenting behavior patterns 
across 7,488 Instagram comments from 77 unique users on 257 photos.

## Key Questions Explored
- Which users are the most active commenters?
- Which photos attract the most engagement?
- How do emoji and hashtag usage vary across users?
- What is the overall sentiment distribution of comments?
